# 接头词

| 接头词 | 接头词含义      | 单词        | 单词解释             |
|--------|-----------------|-------------|----------------------|
| 군-    | 多余、额外       | 군말        | 多余的话、废话        |

| 날-    | 生的、未加工的   | 날고기      | 生肉                 |
|        |                 | 날것        | 生的东西             |

| 덧-    | 额外、重复       | 덧니        | 虎牙                 |
|        |                 | 덧문        | 双层门               |

| 막-     | 最后、末尾、胡乱         | 막내         | 老幺                        |

|        |                 | 외딸        | 独生女               |
| 잔-    | 小的、零碎的     | 잔돈        | 零钱                 |
