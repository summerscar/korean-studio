# 韩文语法：〜(으)ㄹ래요?
## 句型
- 동사 + (으)ㄹ래요?
- 형용사 + (으)ㄹ래요?
## 含义
- 表示请求、建议或征求对方意见
## 例子
1. <Speak>뭐 먹을래요?</Speak> (想吃什么？)
2. <Speak>같이 가고 싶으세요?</Speak> (你想一起去吗？ )
3. <Speak>함께 할래요?</Speak> (你要一起做吗？)
## 注意
- 询问对方意见时使用
- 通常用于非正式场合
## 类似语法
- ~(으)ㄹ까요? (表示对方同意或询问对方打算做什么)
